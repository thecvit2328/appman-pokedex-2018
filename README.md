### Up and running steps:

1.  Clone this repo
    ```
    $ git clone THIS-REPO-URL
    ```
    ```
    $ cd appman-pokedex-2018
    ```
2.  Install NPM packages

    ```
    $ npm install
    ```

3.  Initialize & Start application

    ```
    $ npm run app
    ```

4.  Open your browser at http://localhost:3000
